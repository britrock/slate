##  天天分钱方案详细：share-detail

返回天天分钱方案详细

### 请求URI

`POST http://{domain}/api/share/detail`

### 请求参数
参数 | 默认值 | 类型 | 是否必填 | 描述 
--------- | ------- | --------- | ------- | ----------- 
token| - | String | 是 | 令牌
planId| - | String | 是 | 方案id

        "openCode": [[7,10,6,1,2,5],[1]],                   //开奖号码

        "betMoney":120.00,                          //方案金额
