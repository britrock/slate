##  天天分钱参与方案：share-join

返回天天分钱参与方案结果

### 请求URI

`POST http://{domain}/api/share/join`

### 请求参数
参数 | 默认值 | 类型 | 是否必填 | 描述 
--------- | ------- | --------- | ------- | ----------- 
token| - | String | 是 | 令牌
planId| - | String | 是 | 方案id

```json
{
    "code": 200,                                 //状态码
     "msg":"成功"                               //参与方案结果
}
```
