##  优惠券领取：coupon-get

返回领取结果

### 请求URI

`POST http://{domain}/api/coupon/get`

### 请求参数
参数 | 默认值 | 类型 | 是否必填 | 描述 
--------- | ------- | --------- | ------- | ----------- 
token | - | String | Y | 令牌
couponId | - | Int | Y | 优惠券Id
 
```json
{
    "code": 200,
    "data": null,
    "msg": null
}
```
